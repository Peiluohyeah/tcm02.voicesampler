## Getting Started

To set up a local instance of the application, follow the steps below.  

1.Update to the latest LTS Unity version  
2.Update all packages in the Package Manager  
3.Update all submodules  

### Requirements

#### Hardware
##### Deployment
Nvidia GTX6000  
32Gb RAM  
Enttec DMX USB Pro 70304 Lighting Interface  
etc.
##### Development
Nvidia GTX6000  
32Gb RAM  
Enttec DMX USB Pro 70304 Lighting Interface  
etc.

#### Software
Install npm 
install openupm `$ npm install -g openupm-cli`  
Windows 11 Pro     
Unity 2022.3.18  
MadMapper v.xxx  
etc.  


The following packages are used in compilation.

- JetBrains Rider Editor
- Cinemachine
- ProBuilder
- Test Framework

- Timeline
- Unity Collaborate

- Unity UI
